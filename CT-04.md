# CT-04 - Validação das informações exibidas no extrato

## Cenário

Validar que as transações são apresentadas corretamente no extrato da conta.

## Pré-condição

- O usuário possui uma conta cadastrada.
- O usuário está autenticado.
- A conta possui movimentações registradas.

## Passos

1. Realizar login no sistema.
2. Acessar a tela **Extrato**.
3. Verificar as transações exibidas.
4. Validar as informações apresentadas em cada movimentação.

| Resultado Esperado | Resultado Obtido | Status |
|--------------------|------------------|--------|
| Cada transação deve exibir a data em que foi realizada. | A data foi exibida corretamente. | ✅ Aprovado |
| Cada transação deve exibir o tipo (Abertura de conta, Transferência enviada ou Transferência recebida). | O tipo da transação foi exibido corretamente. | ✅ Aprovado |
| Valores de saída devem aparecer em vermelho e iniciar com o sinal (-). | Não foi possível validar, pois não houve transação de saída. | ⚠️ Não executado |
| Valores de entrada devem aparecer em verde. | Os valores de entrada foram exibidos na cor verde. | ✅ Aprovado |
| Transações sem comentário devem exibir (-). | Não foi possível validar, pois não houve transação sem comentário. | ⚠️ Não executado |
