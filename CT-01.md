# CT-01 - Cadastro realizado com sucesso

## Cenário
Validar que um usuário consegue criar uma conta preenchendo todos os campos obrigatórios corretamente.

## Pré-condição

- O sistema de cadastro está disponível.
- O usuário está na tela de cadastro.
- O e-mail informado ainda não está cadastrado.

## Passos

1. Acessar a tela de cadastro.
2. Preencher o campo **Nome** com um nome válido.
3. Preencher o campo **Email** com um e-mail válido.
4. Preencher o campo **Senha**.
5. Preencher o campo **Confirmação de senha** com a mesma senha informada anteriormente.
6. Ativar ou manter desativada a opção **Criar conta com saldo**.
7. Clicar no botão **Cadastrar**.

> **Observação:** Durante a execução foi identificado que o botão de alternância do saldo não responde ao clique no círculo branco. A opção só é ativada/desativada ao clicar na barra.

| Resultado Esperado | Resultado Obtido | Status |
|--------------------|------------------|--------|
| O cadastro é realizado com sucesso. | O cadastro foi realizado com sucesso. | ✅ Aprovado |
| O sistema exibe o número da conta criada. | O número da conta foi exibido corretamente. | ✅ Aprovado |
| Com a opção "Criar conta com saldo" ativada, a conta deve ser criada com saldo de **R$ 1.000,00**. | A conta foi criada com saldo de **R$ 1.000,00**. | ✅ Aprovado |
| Com a opção "Criar conta com saldo" desativada, a conta deve ser criada com saldo de **R$ 0,00**. | A conta foi criada com saldo de **R$ 0,00**. | ✅ Aprovado |
