# bugbank-testes-manuais
Testes feitos no site Bugbank da EBAC como proposta de atividade

Projeto de Testes Manuais - BugBank (EBAC)
Sobre o projeto

Este repositório contém casos de teste manuais desenvolvidos como atividade prática do curso de QA da EBAC utilizando a aplicação BugBank.

O objetivo foi validar funcionalidades de:

Cadastro de usuários
Validação de campos obrigatórios
Exibição de saldo no extrato
Aplicação testada
Sistema: BugBank
Ambiente: https://bugbank.netlify.app/
Tipo de teste: Teste manual funcional
Metodologia: Baseada em cenários de uso e validação de requisitos
Casos de teste
ID	Funcionalidade	Objetivo
CT-01	Cadastro	Validar cadastro com sucesso
CT-02	Cadastro	Validar mensagens de erro e campos obrigatórios
CT-03	Extrato	Validar exibição do saldo disponível
